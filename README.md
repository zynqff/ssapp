# 📖 Сборник Стихов — мобильное приложение

Flutter-приложение для хранения, чтения и управления коллекцией стихотворений с поддержкой офлайн-режима и ИИ-чата.

**Версия:** 1.0.4 
**Платформа:** Android (Flutter)  
**Backend:** [ssback-go.onrender.com](https://ssback-go.onrender.com)

---

## ✨ Возможности

- 📚 **Библиотека стихов** — просмотр, поиск и фильтрация стихотворений
- 🔖 **Закреплённое стихотворение** — одно стихотворение всегда под рукой
- ✅ **Отметка прочитанного** — отслеживание прогресса чтения
- 🤖 **ИИ-чат** — диалог с ИИ по теме поэзии (требует ключ доступа)
- 🌙 **Светлая/тёмная тема** с выбором акцентного цвета
- 📶 **Офлайн-режим** — все данные кэшируются локально в SQLite
- 🔄 **Фоновая синхронизация** — очередь действий при отсутствии сети
- 🔐 **Авторизация** через логин/пароль или Google Sign-In
- 🛡️ **Режим администратора** — добавление, редактирование и удаление стихов

## 🚀 Запуск проекта

### Требования

- Flutter SDK `^3.4.0`
- Dart SDK `^3.4.0`
- Android Studio / VS Code с плагином Flutter
- Устройство или эмулятор Android

### Установка

```bash
# Клонировать репозиторий
git clone <repo-url>
cd ssapp

# Установить зависимости
flutter pub get

# Запустить приложение
flutter run
```

```bash
flutter build apk --release
```

## 🔌 API

Все запросы идут на `https://ssback-go.onrender.com`.

> ⚠️ Сервер размещён на Render (free tier) и может «засыпать». Первый запуск может занять до 30 секунд.

### Основные эндпоинты

| Метод | Путь | Описание |
|---|---|---|
| `POST` | `/api/login` | Вход по логину и паролю |
| `POST` | `/api/register` | Регистрация нового пользователя |
| `POST` | `/api/google/mobile-auth` | Вход через Google |
| `GET` | `/api/me` | Получение профиля текущего пользователя |
| `GET` | `/api/poems` | Список всех стихотворений |
| `POST` | `/api/toggle_read` | Отметить/снять отметку "прочитано" |
| `POST` | `/api/toggle_pin` | Закрепить/открепить стихотворение |
| `POST` | `/api/ai/chat` | Отправить сообщение ИИ |

## 🔒 Аутентификация

Приложение поддерживает два способа входа:

1. **Логин + пароль** — минимальная длина пароля 4 символа
2. **Google Sign-In** — через аккаунт Google

JWT-токен сохраняется в `FlutterSecureStorage` и автоматически подставляется в каждый запрос.

---

## 📶 Офлайн-режим

Приложение работает без интернета:

- Стихи хранятся локально в SQLite
- Действия пользователя (прочитано, закреплено) добавляются в очередь синхронизации
- При восстановлении соединения очередь автоматически отправляется на сервер
