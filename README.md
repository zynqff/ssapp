# 📖 Сборник Стихов — мобильное приложение

Flutter-приложение для хранения, чтения и управления коллекцией стихотворений с поддержкой офлайн-режима и ИИ-чата.

**Версия:** 1.0.6  
**Платформа:** Android (Flutter)  
**Backend:** 

---

## ✨ Возможности

- 📚 **Библиотека стихов** — просмотр, поиск и фильтрация стихотворений
- 🔖 **Закреплённое стихотворение** — одно стихотворение всегда под рукой
- ✅ **Отметка прочитанного** — отслеживание прогресса чтения
- 🤖 **ИИ-чат** — диалог с ИИ по теме поэзии (требует ключ доступа)
- 🌙 **Светлая/тёмная тема** с выбором акцентного цвета
- 📶 **Офлайн-режим** — все данные кэшируются локально в SQLite
- 🔄 **Фоновая синхронизация** — очередь действий при отсутствии сети
- 🔐 **Авторизация** через OTP на email или Google Sign-In
- 🛡️ **Режим администратора** — добавление, редактирование и удаление стихов

## 🚀 Запуск проекта

### Требования

- Flutter SDK `^3.4.0`
- Dart SDK `^3.4.0`
- Android Studio / VS Code с плагином Flutter
- Устройство или эмулятор Android

### Установка

```bash
# Клонировать репозиторий
git clone <repo-url>
cd ssapp

# Установить зависимости
flutter pub get

# Запустить приложение
flutter run
```

### Сборка APK

```bash
flutter build apk --release
```

## 🔒 Аутентификация

Приложение поддерживает два способа входа:

1. **OTP на email** — одноразовый код, отправляется на почту
2. **Google Sign-In** — через аккаунт Google

Сессия управляется через Supabase Auth и автоматически подставляется в каждый запрос.

---

## 📶 Офлайн-режим

Приложение работает без интернета:

- Стихи хранятся локально в SQLite
- Действия пользователя (прочитано, закреплено) добавляются в очередь синхронизации
- При восстановлении соединения очередь автоматически отправляется на сервер
